
# MindPulse

MindPulse is an AI-powered mental wellness platform designed for students. It helps users track mood, write journals, and get AI-based insights for better mental health.

## Getting Started

Run the development servers:

### Backend
```bash
cd backend
npm install
npm run dev
Frontend
cd frontend
npm install
npm run dev
Open http://localhost:5173 in your browser.

AI Service
cd ai_service
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
Service runs on http://localhost:5001

Features
User Authentication

Mood Tracking

Daily Journaling

AI Sentiment Analysis

Mood Dashboard

AI Wellness Chatbot

Stress Management Tools

Emergency Support

Deployment
Frontend can be deployed on Vercel and backend services on Heroku, Railway, or Render.
